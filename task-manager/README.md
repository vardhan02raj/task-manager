# 🚀 Task Manager App

A simple full-stack Task Manager application built using React (frontend) and Node.js (backend).

---

## 📥 Clone the Repository

```bash
git clone https://github.com/vardhan02raj/task-manager.git
cd task-manager
```

---

## ⚙️ Install Dependencies

### Frontend

```bash
npm install
```

---

### Backend

```bash
cd backend
npm install
cd ..
```

---

## ▶️ Run the Project

### 🟢 Start Backend Server

Open terminal:

```bash
cd backend
node server.js
```

Server runs at:
http://localhost:5000

---

### 🟢 Start Frontend

Open another terminal:

```bash
cd task-manager
npm run dev
```

Frontend runs at:
http://localhost:5173

---

## 🔗 API Endpoints

| Method | Endpoint   | Description   |
| ------ | ---------- | ------------- |
| GET    | /tasks     | Get all tasks |
| POST   | /tasks     | Add new task  |
| PUT    | /tasks/:id | Update task   |
| DELETE | /tasks/:id | Delete task   |

---

## ⚠️ Important Notes

* Backend must be running before frontend
* Data is temporary (stored in memory)
* Data resets when server restarts

---

## 🛠️ Tech Stack

* React (Vite)
